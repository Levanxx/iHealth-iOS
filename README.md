# iHealth para iOS

Aplicación de salud personal desarrollada con SwiftUI.

## Funcionalidades

- Registro e inicio de sesión con credenciales protegidas por Keychain.
- Datos de salud separados por usuario.
- Medicamentos, citas, síntomas e información médica.
- Recordatorios locales mediante UserNotifications.
- Contactos médicos y apertura de WhatsApp.
- Asistente conversacional integrado con Voiceflow.
- Temas claro, oscuro y automático.
- Interfaz con degradados morados y animaciones adaptadas a iOS.

## Requisitos

- Xcode 26 o posterior.
- iOS 17 o posterior.

Abre `iHealthIOS.xcodeproj` y ejecuta el esquema `iHealthIOS`.
